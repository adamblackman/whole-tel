       - If mismatch, return { success: false, error: 'Split amounts must equal the booking total exactly.' }
       - Upsert to booking_splits using admin client: rows mapped from splits with booking_id, invitation_id, amount, payment_status: 'unpaid'. Use onConflict: 'booking_id,invitation_id'

       - Update split row with stripe_payment_link_id and stripe_payment_link_url

       }
