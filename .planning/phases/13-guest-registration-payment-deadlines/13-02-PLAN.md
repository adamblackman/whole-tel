Add a helper function at the top of the file (or inline):
```typescript
function computeActivityDeadline(checkIn: string, createdAt: Date): string {
  const checkInMinus30 = new Date(new Date(checkIn).getTime() - 30 * 24 * 3600 * 1000)
  const createdPlus7 = new Date(createdAt.getTime() + 7 * 24 * 3600 * 1000)
  // Use whichever is earlier, but never in the past
  const deadline = checkInMinus30 < createdPlus7 ? checkInMinus30 : createdPlus7
  return deadline.toISOString()
}
```

Important: The `booking.insert()` call returns `booking.id`. Use the same `supabase` client (user-scoped) for the update since the user owns the booking. The update must happen BEFORE the `redirect(stripeUrl)` call.

The `select('id')` on the insert should be expanded to `select('id, created_at')` so we can use the actual `created_at` from the database:
```typescript
.select('id, created_at')
```
Then use `new Date(booking.created_at)` instead of `new Date()` for precise deadline computation.
