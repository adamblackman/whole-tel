Create `src/lib/pricing.ts` as a pure function module with NO framework dependencies:

1. Export `PricingInput` interface with fields: nightlyRate, cleaningFee, nights, guestCount, guestThreshold (number | null), perPersonRate (number | null), selectedAddOns array (each with id, name, price, pricingUnit, includedGuests, perPersonAbove)
2. Export `PricingBreakdown` interface with fields: accommodationSubtotal, perPersonSurcharge, surchargeDetail (or null), addOnItems array (each with id, name, baseCost, tierCost, totalCost, tierDetail or null), addOnsTotal, cleaningFee, processingFee, total
3. Export `calculatePricing(input: PricingInput): PricingBreakdown` pure function:
   - accommodationSubtotal = nightlyRate * nights
   - perPersonSurcharge: if guestThreshold AND perPersonRate are both non-null AND guestCount > guestThreshold, then (guestCount - guestThreshold) * perPersonRate * nights, else 0

   - total = accommodationSubtotal + perPersonSurcharge + cleaningFee + addOnsTotal + processingFee
