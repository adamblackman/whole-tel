2. **Construct disabled dates** — respect half-open `[)` interval:
   ```typescript
   const bookings = bookingsResult.data ?? []
   const disabledDates = bookings.map((b) => ({
     from: new Date(b.check_in),
     to: new Date(new Date(b.check_out).getTime() - 86400000), // day before checkout
   }))
   ```
   The subtraction of 86400000ms (1 day) is critical: the schema uses `[)` half-open bounds, meaning the checkout day is NOT part of the booked range. Without this, guests couldn't book back-to-back stays.

   ```tsx
       disabledDates={disabledDates}

   </div>
   ```
