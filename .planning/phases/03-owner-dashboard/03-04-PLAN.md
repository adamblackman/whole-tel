Props:
```typescript
interface BookingsTableProps {
  bookings: Array<{
    id: string
    check_in: string
    check_out: string
    guest_count: number
    total: number
    status: 'pending' | 'confirmed' | 'cancelled'
    created_at: string

    }
  }>
```

Note: The Supabase query uses `properties!inner(id, name, owner_id)` which returns a single object (not array) because it's a FK relationship. The `!inner` ensures only bookings where the joined property matches the filter are returned.

  - pending: `<Badge variant="secondary">Pending</Badge>`
  - cancelled: `<Badge variant="destructive">Cancelled</Badge>`
- "Booked" column: format created_at as relative or short date
- If bookings array is empty: show "No bookings yet" message instead of an empty table
